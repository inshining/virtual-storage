## 개요

본 프로젝트에서 백엔드 API 서버만을 구현합니다. 
본 프로젝트 파일 업로드, 다운로드, 삭제, 목록 조회 등의 기능을 제공합니다. 또한, 폴더로 파일을 관리할 수 있습니다.


## 기술 스택

- 백엔드: Java(22), Spring Boot(3.3.2)
- 데이터베이스: MySQL
